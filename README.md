# Coin Town

Coin Town — это онлайн-игра, в которой пользователи могут создавать аккаунты, зарабатывать коины, покупать домики и испытывать удачу в рулетке. Каждый игрок получает ежедневную награду в размере 10 коинов каждые 24 часа.
На сайте реализована система сохранения данных, благодаря которой прогресс игроков не теряется. Все никнеймы уникальны и не могут повторяться. Пользователи могут входить в свои аккаунты и выходить из них в любое время.
Администраторы имеют специальные возможности для управления игровым миром, включая создание и удаление домиков.
Сайт доступен по адресу: https://coin-town.ru

## основные функции

### админ-панель
 - управления пользователями;
 - удаление домиков;
 - настройка сайтов.

### домики
- покупка домиков;
- просмотр своих домиков;
- продажа домиков.

### рулетка
 - получения случяйных наград;
 - выигрыш монет и бонусов

### пользователи
 - регистрация акаунта;
 - вход в акаунт;
 - уникальные ники без повторений.

### дизайн
 - современый интерфейс;
 - поддержка телефонов и компьютеров;
 - тёмная и белая тема.

## цель проекта

создать интересную игровую платформу для пользователей сайта Coin-Town.ru.
